## What is OpenTag?

An open-ended, attribute-based tagging system for LoL esports analysis. Instead of fixed categories, every entity (player, team, champion, duo, matchup) is tagged with granular, data-driven attributes that combine to generate betting signals.

**Key numbers:**
- **218** entities across 7 types (player, team, champion, duo, matchup, matchup_champ, pattern)
- **4,506** tags across 40 prefixes (combat, meta, laning, vision, build, rank, ...)
- **141** auto-discovered patterns (correlations, conditional WR, combinations, win conditions)
